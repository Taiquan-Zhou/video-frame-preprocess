# Changelog

All notable changes to this project are documented here.

## 0.1.0

- Extracted `video-frame-preprocess` from the original LingBot Map video preprocessing module.
- Added single-video frame extraction and conservative quality filtering.
- Added batch processing.
- Added run manifests, frame reports, selected-frame mappings, and summaries.
- Added optional clean filtered video output.
- Added artifact health checks.
- Added Chinese and English documentation.
